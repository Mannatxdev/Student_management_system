# Student_management_system
in this we can add,view,update,delete and store data in database


## Steps to make system ready
1. python -m venv venv(name of venv)
2. venv\scripts\activate
3. pip install uv
4. uv pip install django


## Create Django Project (Main Folder)
1. django-admin startproject config .

## Create App
1. python manage.py startapp students
